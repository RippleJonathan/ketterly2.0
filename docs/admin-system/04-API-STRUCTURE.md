# API Structure & Patterns

Defines the API client architecture, patterns, and functions for interacting with Supabase.

Includes:
- Supabase client setup (browser, server, admin service role)
- Standard ApiResponse and PaginatedResponse patterns
- Example Lead API functions: getLeads, getLeadById, createLead, updateLead, deleteLead, assignLead, updateLeadStatus, convertLeadToCustomer, getLeadPipelineStats
- Quote API examples and more
- React Query hooks examples in `src/hooks/`
